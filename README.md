# Event Management System

## Description

This is a simple event management system that allows users to create, view, update and delete events. The system is
built using Spring boot and Angular.

### Usage

1. Database:
   - docker-compose up -d


2. Backend:
    - cd server
    - mvn spring-boot:run
   

3. Frontend:
    - cd client
    - ng serve
